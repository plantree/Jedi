## Swagger本地接口协同编辑

#### 功能

本地运行Swagger Editor。因为Swagger网站在线服务国内访问速度较慢，体验不佳。而且乞丐版是Readonly，无法协同编辑，因此在本地起一个服务，便于协作。

#### 工具安装

- npm install http-server -g
- swagger-editor.zip
- jedi.yaml

#### 使用

- http-server swagger-editor-3.14.8

#### 参考：
- https://github.com/swagger-api/swagger-editor/releases/
